# 💬 Random Quote Generator

An advanced random quote generator built with HTML, CSS, and JavaScript as part of the **Horizon TechX App Development Internship**.

## 🚀 Features
- 🔀 Random quote on every click
- 📂 Category filter (6 categories, 30 quotes)
- 🎨 Background color changes per category
- ✨ Smooth fade-in animation
- ❤️ Save favorite quotes
- 📋 Copy quote to clipboard
- 🔊 Text-to-speech (read out loud)
- 🐦 Twitter & 💬 WhatsApp share buttons
- 💾 localStorage (favorites saved)

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Speech API
- localStorage

## ▶️ How to Run
1. Download `random-quote-generator.html`
2. Open in any browser — no installation needed!
